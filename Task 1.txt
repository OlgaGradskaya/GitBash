1.Определить имя папки, в которой вы находитесь - pwd
2.Создать папку с именем test1 - mkdir test1
3.Перейти в папку test1 - cd test1
4.Создать файл 1,2 и 3 внутри каталога test1 -touch cd test1 touch 1 2 3
5.Проверить содержимое каталога test1 - ls -la
6.Создать папку test2 внутри домашней директории - mkdir ~/test2
7.Удалить папку test2 - rmdir test2
8.Удалить файл 2- rm 2
9.Создать папку test3 и добавить в нее два файла - mkdir test3 && touch test3/1 test3/2
10.Удалить папку test3 - rm -rf test3
11.Создать папку test4 - mkdir test4
12.Переместить файлы 1 и 3 в папку test4 - mv 1 3 test4/
13.Добавить в файл 1 три строки со словами line - echo line >> 1 && echo line >> 1 && echo line >> 1
14.Посмотреть содержимое файла 1 - cat 1
15.Добавьте в файл 3 три строки со словами line - echo line >> 3 && echo line >> 3 && echo line >> 3
16.Просмотрите содержимое двух файлов (1 и 3) сразу - cat 1 3
17.Используя один из редакторов замените все строки - зайти в редактор vim файла 1(vim 1),для редактирования файла нажать i,отредактировать три строки (добавляем порядковый номер: line1 line2 line3),нажать ESC. Пишем :wq. Тоже самое делаем для файла 3
